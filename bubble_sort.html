Bubble Sort (Sắp xếp nổi bọt)

1. Giới thiệu

  • Bubble Sort là thuật toán sắp xếp đơn giản. Nó lặp qua danh sách, so sánh các phần tử liền kề, và hoán đổi nếu chúng không theo thứ tự. Quá trình lặp lại cho đến khi danh sách được sắp xếp.

2. Code

Normal Function

        function bubbleSort(arr) {
            let n = arr.length;
            for(let i = 0; i < n - 1; i++) {
                for(let j = 0; j < n - 1 - i; j++) {
                    if(arr[j] > arr[j + 1]) {
                        let temp = arr[j];
                        arr[j] = arr[j + 1];
                        arr[j + 1] = temp;
                    }
                }
            }
            return arr;
        }
        
        const arr = [64, 34, 25, 12, 22, 11, 90];
        console.log("Array: ", arr);
        
        const sortedArr = bubbleSort(arr);
        console.log("Array after sorted: ", sortedArr);

3. Kết Luận

  • Bubble Sort dễ hiểu, phù hợp với mảng nhỏ.
  • Độ phức tạp: O(n^2) trong trường hợp xấu nhất.
  • Không hiệu quả với dữ liệu lớn, nhưng hữu ích để học các khái niệm cơ bản.
